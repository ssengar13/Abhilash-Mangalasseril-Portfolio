# Custom Portfolio Website

Welcome to the **Custom Portfolio Website**! This project showcases various businesses and provides an admin portal for easy management.

## Features

- **Responsive Design:** The website is fully responsive, ensuring a seamless experience on all devices.
- **Admin Portal:** Users can add, edit, and delete businesses and posts through an intuitive admin interface.
- **User-Friendly Interface:** A modern and clean design for easy navigation for both visitors and admin users.

## Technologies Used

- **HTML5**: For structuring the web pages.
- **CSS3**: For styling the web pages.
- **JavaScript**: For enhancing user interaction.
